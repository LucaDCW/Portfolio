Site Entreprise IT - Capgemini

Description du Projet

Site web vitrine développé pour présenter les activités et services de l'entreprise française Capgemini. Ce projet met l'accent sur un design moderne et une présentation professionnelle des services de conseil en technologies de l'information.

Le site adopte une mise en page en "zig-zag" qui guide naturellement le regard de l'utilisateur à travers les différentes sections. Cette conception permet une présentation dynamique et engageante des activités de l'entreprise.

Savoirs Acquis

  • Structure HTML5 simple
  • Mise en page CSS3
  • Principes de design web moderne
  • Interface utilisateur en zig-zag
  • Optimisation des images et médias web

Savoir-faire Développés

  • Intégration de contenu multimédia
  • Gestion efficace des versions avec Git
  • Déploiement automatisé sur GitHub Pages
